Gestor de base de datos

Este proyecto consistia en que a través de de comandos personalizados el usuario pudiera manipular las bases de datos existentes en su sistema. Para lo cual se utilizo un arbol binario de decisiones en el cual se realizaba la busqueda de palabras claves. Después de esto si la sintaxis era correcta el sistema podria llegar a omastrar desde los registros de la base de datos hasta las propiedades de cada una de las tablas. Cabe mencionar que para mejorar la experiencia de usuario, en este caso los comandos fueron puestos en español, para que así los uduarios de habla hispano tuvieran mejor comprensión al manejar sus base de datos.

Control Veterinario

El presente se propuso para resolver la problematica que muchas veterinaria enfrentan, la oranización de citas , asi como tambien brindar informes al dueño de la mascota acerca de o que esta tiene. Con el presente sistema el dueño o encargado de las mascotas puede estar al tanto de los idagnosticos, asi como tambien cuando le toca la proxima cita, con quien y para que.
link al repositorio.

Salesian Life

Este proyecto fue pensado en si para los niños más pequeños del "Colegio Salesiano Santa Cecilia". Es un juego el cual en si le enseña a los niños la vida de Don Bosco de una manera interactiva y divertida, asi como también sirve para que los maestros puedan ver el avance de estos según la historia y asi poder brindar una ponderación a dicha actividad. Este juego fue desarrollado con Unity y C# (esto en la parte de escritorio), mientras que el sistema de calificaciones fue diseñado para que los profesores lo pudieran registrar a través de un sistema web.
